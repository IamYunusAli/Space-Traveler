## Getting Started

**How to clone the project and get started**

To Run the project

```
1. git clone https://github.com/IamYunusAli/Space-Traveler.git
2. cd Space-Traveler
3. npm i
4. npm run start
5. use http://localhost:3000/ on your browser
```
To Run test
 ``` 
 npm test
 ``` 
